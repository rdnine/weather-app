# Weather App - Client Side
**Weather App UI**

---
## Tech Stack
- Vue 3
- Vuex 4
- Axios
- PrimeVUE
- ChartJS

---
## Install

**Considering you are already inside the "client" folder**

```
  $ npm install
  $ npm run build
```

## Run
```
  $ npm run start
```

---

## Requirements

This app requires the web server on the project folder "server"

**First follow the given instructions there**
